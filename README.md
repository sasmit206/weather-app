A basic weather application that fetches real-time weather data for any city using an external Weather API. The app dynamically appends the city name entered by the user to the API request and displays the fetched weather details on the front end.


*Features*

Search weather by city name

Real-time weather data fetching using an API

Displays temperature and weather conditions

Clean and responsive UI

API key securely handled (not exposed in the repository)


*Tech Stack*

Frontend

HTML – Structure of the application

CSS – Styling and layout

JavaScript (Vanilla JS) – Handles API requests, DOM manipulation, and user interactions
